Фишка с подчеркнутой первой буквой в любом меню. Если её нажать то вызовется соответствующий пункт меню
Также большая часть программ и файлов открывается по первой букве
# Система
Win + текст - поиск по системе
Win + номер на панели задач - быстрый старт закрепленного приложения
Win + t - фокус на панели задач
Alt + F4 - закрыть окно
Win + b - фокус на трее
Shift + F10 или клавиша рядом с правым CTRL - вызов контекстного меню
Alt - фокус на верхнем меню
Ctrl + alt + tab - панель открытых окон перестаёт сворачиваться и через delete можно удобно позакрывать все ненужные окна
Alt + space - меню окна
Alt + space + c(рус раскл) / Ctrl + m - свернуть активное окно
# Виртуальные рабочие столы
Ctrl + win + d - создать доп рабочий стол
Ctrl + win + left/right - переключение между раб столами
Ctrl + win + f4 - закрыть доп рабочий стол
# Проводник
Win + e - открыть проводник
Ctrl + w - закрыть проводник
Ctrl + f - поле поиска
Ctrl + l - поле адреса
Ctrl + shift + n - новая папка
Ctrl + n - дубль открытого окна
Alt + enter - открыть свойства папки/файла
Ctrl + pgup/pgdn - переключение вкладок
Alt + up - подняться на уровень выше
Alt+ left/right - история перемещения по папкам
Ctrl + space - выделение только нужных элементов
# Браузер
Ctrl + t - новая вкладка
Ctrl + n - новая вкладка в соседнем окне
Alt+ left/right - история перемещения по страницам внутри вкладки
Ctrl + l потом alt + enter - дубликат вкладки
Переходить по ссылкам можно через поиск ctrl + f потом enter
