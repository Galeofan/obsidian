На первых порах можно использовать псевдокод и блок-схемы:
```java
проснулся;

if (выходной?) гуляю;
else           иду на работу;

пошел спать;
```

```java
Объявили A, B, C;  
  
if (A > B)      проверить (А > C);  
    if (A > C)  вывести на экран А;  
    else        вывести на экран C;  
else if (B > C) вывести на экран B;  
else            вывести на экран C;  
  
закончить программу;
```
