Монитор - объект синхронизации(хранит состояние можно или нельзя обратиться к сущности). Монитором может быть любой класс или объект.
Монитор можно передать с помощью ключевого слова synchronized. 
Пример:

```java
public class Counter {  
  
    private int value;  
    private int value2;  
  
    private Object monitor1 = new Object();  
    private Object monitor2 = new Object();  
  
    public void inc() {  
        synchronized (monitor1) {  
            value++;  
        }  
    }  
  
    public void dec() {  
        synchronized (monitor1) {  
            value--;  
        }  
    }  
  
    public int getValue() {  
        return value;  
    }  
  
    public void inc2() {  
        synchronized (monitor2) {  
            value2++;  
        }  
    }  
  
    public void dec2() {  
        synchronized (monitor2) {  
            value2--;  
        }  
    }  
  
    public int getValue2() {  
        return value2;  
    }  
}
```

Если добавить synchronized в сигнатуру метода то в качестве монитора используется тот объект у которого вызван данный метод
Пример(записи равнозначны):
```java
    public synchronized void inc() {    
            value++;  
        }  
    }  
    
    public void inc() {  
        synchronized (this) {  
            value++;  
        }  
    }  
```

Мониторы внутри одного объекта могут быть разными как в примере выше, именно для этого и добавлена возможность делать их по объектам
