# Настройки и бинды
- Отображать номера строк - вкл
- Ограничить максимальную длину строки - выкл
- Забиндить стандартные хоткеи:
	- Alt + 1 - "Файловый менеджер: Показать файловый менеджер"

- Забиндить хоткеи гита как в идее:
	- CTRL + K - коммит
	- CTRL + SHIFT + K - push
	- CTRL + SHIFT + O - fetch
	- CTRL + SHIFT + U - pull
