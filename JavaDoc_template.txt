 * @param input описание параметра (например, входная строка)

 Пример:
 /**
 * Проверяет, что строка не пустая и не состоит только из пробелов.
 *
 * <p>Возвращает {@code true}, если строка содержит хотя бы один
 * непустой символ, иначе {@code false}.</p>
 *
 * <pre>
 * Input           | Result
 * ----------------+--------
 * null            | false
 * ""              | false
 * "   "           | false
 * "abc"           | true
 * " abc "         | true
 * </pre>
 *
 * @param input строка для проверки
 * @return true, если строка содержит непустые символы
 */
public static boolean hasText(String input) {
    return input != null && !input.trim().isEmpty();
}
