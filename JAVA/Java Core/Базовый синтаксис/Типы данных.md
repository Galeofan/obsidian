В Java все типы данных можно разделить на две группы: примитивные и ссылочные типы:

Примитивные типы:
**byte** – используется для хранения байтовой информации в памяти. 8 bits.
**short** – целочисленный тип. 16 bits
**int** – целочисленный тип. 32 bits
**long** – целочисленный тип. 64 bits
**float** – числа с плавающей точкой. 32 bits
**double** – числа с плавающей точкой. 64 bits
**boolean** – описывают логический тип. 1 bit
**char** – символьный тип. 16 bits
