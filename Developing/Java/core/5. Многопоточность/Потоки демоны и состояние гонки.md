Есть 2 вида потоков:
1) Основные(важные) - обязательно должны завершить свою работу для того чтобы главный поток мог завершиться
2) Потоки-демоны(неважные) - необязательно должны завершаться чтобы главный поток мог завершиться

!!!Для потоков-демонов нужно вызывать thread.setDaemon(true) чтобы при завершении основных потоков поток-демон завершился автоматически!!!

Потоки-демоны завершают работу только в том случае когда все основные потоки закончат свою работу, а не только главный

Состояние гонки(Race Condition) - ситуация когда разные потоки обращаются одновременно к одной и той же несинхронизированной переменной, результат при этом получается неожиданный в зависимости от того какой поток положит результат первым
