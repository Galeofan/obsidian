### 1. Перемещение по строке
- Alt + B - переместить курсор к началу слова слева
- Alt + F - переместить курсор к началу слова справа
- Ctrl + A — перейти в начало строки.
- Ctrl + E — перейти в конец строки.
### 2. Редактирование текста
- **Ctrl + U** — удалить всё от курсора до начала строки.
- **Ctrl + K** — удалить всё от курсора до конца строки.
- **Ctrl + W** — удалить слово слева.
- **Alt + D** — удалить слово справа.
- **Ctrl + Y** — вставить последнее удалённое (аналог быстрого undo на уровне текста).

### 3. Управление процессами
- **Ctrl + C** — прервать текущее выполнение.
- **Ctrl + D** — выход из текущей оболочки или сигнал "EOF".
- **Ctrl + Z** — отправить процесс в фон (suspend).
- 
Дополнительно:
- **fg** — вернуть процесс из фона.
- **bg** — продолжить выполнение в фоне.

### 4. Навигация по истории
Ускоряет работу с командами, которые ты уже выполнял.
- **Ctrl + R** — поиск по истории команд (инкрементальный).  Пример: набираем часть команды — история сразу подставит совпадение.
- **Ctrl + G** — выйти из режима поиска.
- **Alt + .** — вставить последний аргумент предыдущей команды (очень мощный хоткей).
**Пример:**
```bash
cp /long/path/file.txt . vim <Alt+.>
```
