Выполняем из cmd:
Загрузка образа - docker pull quay.io/keycloak/keycloak:latest
Стоп контейнера - docker stop image-name
Статусы контейнеров - docker ps
Загрузить оффлайн образ из архива - docker load -i keycloak_26.2.3_phone-2.5.0.tar

Запуск keycloak с встроенной бд с сохранением данных:
```powershell
docker run -p 8080:8080 -e KEYCLOAK_ADMIN=admin -e KEYCLOAK_ADMIN_PASSWORD=admin -v "C:\Users\ACuprun\kc-data:/opt/keycloak/data" -v "C:\Users\ACuprun\kc-providers:/opt/keycloak/providers" quay.io/keycloak/keycloak:21.0.2 start-dev --features=declarative-user-profile --spi-phone-default-service=dummy --spi-phone-enabled=true
```
