Всё программирование сводится к хранению, передаче и изменению данных.
Для этих операций в программировании есть понятие [**переменная**](JAVA/Java%20Core/Базовый%20синтаксис/Термины.md#Переменная).

Для того, чтобы в Java создать переменную, необходимо использовать следующий шаблон:
[ТИП ДАННЫХ](JAVA/Java%20Core/Базовый%20синтаксис/Типы%20данных.md) ИМЯ_ПЕРЕМЕННОЙ = ЗНАЧЕНИЕ_ПЕРЕМЕННОЙ;
