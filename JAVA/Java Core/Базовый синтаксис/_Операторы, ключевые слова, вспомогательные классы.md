### Ключевые слова
return с аргументами - указывает, значение какой переменной нужно считать результатом вычисления метода
return без аргументов - мгновенно передать управление программой вызывающему методу

### Классы
####  java.lang.Math
Math.sqrt(a) - корень квадратный от a.
Math.pow(a, b) - возведение числа а в степень b.
Math.round(d) - округление до ближайшего челого числа.

#### java.lang.String
Текстовые данные в Java помещаются в двойные кавычки.
Конкатенация с любым типом данных через оператор +
