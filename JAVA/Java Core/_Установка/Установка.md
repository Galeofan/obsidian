### Основные инструменты

1. Качаем и ставим [JDK](JAVA/Java%20Core/Базовый%20синтаксис/JDK.md) c сайта [Oracle](https://www.oracle.com/java/technologies/downloads/)
2. Качаем и ставим [Intellij IDEA Community Edition](Intellij%20IDEA/Intellij%20IDEA.md) с сайта [Jetbrains](https://www.jetbrains.com/idea/download/?section=windows)
3. Качаем и устанавливаем [__GIT](GIT/__GIT.md): [Тык](https://gitforwindows.org/)
4. Качаем и устанавливаем [GitHub Desktop](GIT/GitHub.md) (необязательно, но удобная штука)
5. В приложухе GitBash генерим связку ключей командой: `ssh-keygen -t rsa -b 4096 -C "your_email@example.com"`, где вместо `your_email@example.com` указываем свою почту из профиля github
6. Командой `eval "$(ssh-agent -s)"` чекаем что клиент для работы через SSH запущен: 
7. Грузим ключи в агента SSH: ssh-add ~/.ssh/id_rsa
8. Копируем в буфер публичный ключ и грузим его в GitHub: clip < ~/.ssh/id_rsa.pub
9. Авторизуемся в Git: ssh -T git@github.com

### Команды для проверки версии Java

Для проверки версии Java можно использовать 2 команды, их вывод примерно аналогичен:

 ```bash
	ls /Library/Java/JavaVirtualMachines
 ```

```Shell
	java --version
```
