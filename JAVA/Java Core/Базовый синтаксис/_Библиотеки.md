### Библиотеки
####  java.lang.Math
- <font color="#31859b">Math.sqrt(a)</font> - корень квадратный от a.
- <font color="#31859b">Math.pow(a,b)</font> - возведение числа а в степень b.
- <font color="#31859b">Math.round(d)</font> - округление до ближайшего челого числа.

#### java.lang.String
Текстовые данные в Java помещаются в двойные кавычки.
Конкатенация с любым типом данных через оператор <font color="#00b050">+</font>

#### org.junit.jupiter.api.Test
Над каждым методом стоит [аннотация](JAVA/Java%20Core/Базовый%20синтаксис/_Термины.md#Аннотация) <font color="#ffff00">@Test</font>. Аннотация позволяет пометить метод, класс или переменную для дополнительной обработки.
Аннотация <font color="#ffff00">@Test</font> позволяет запустить каждый [метод](JAVA/Java%20Core/Базовый%20синтаксис/_Термины.md#Метод) по отдельности. В случае с тестированием это удобный механизм.
- <font color="#31859b">assertThat()</font> -  метод который сравниваетт два значения: результат вычисления и ожидаемый результат. В качестве аргумента мы указываем выходное значение - output, 
- <font color="#31859b">isEqualTo()</font> - метод в который мы передаем ожидаемое значение - expected, 
- <font color="#31859b">withPrecision()</font> - метод в который передаем точность сравнения - value, которую мы указываем в качестве аргумента метода .
```java
void whenConvert140RblThen2Euro() {  
    float input = 140; //Входное значение
    float expected = 2;  //Ожидаемое значение
    float output = Converter.rubleToEuro(input);  //Вызов проверяемого метода
    float value = 0.0001f; //Точность (до десятитысячных)
    assertThat(output).isEqualTo(expected, withPrecision(value));  //Сравнение
}
```
