Для написания модульных тестов как правило используется специальный [фреймворк](JAVA/Java%20Core/Базовый%20синтаксис/_Термины.md#Фреймворк). Например JUnit 5.

У нас в проекте используется система Maven, вся конфигурация проекта находится в файле pom.xml, его вы можете найти в корне проекта. Благодаря наличию этого конфигурационного файла, произошло автоматическое подключение фреймворка JUnit5 к проекту. Термин "подключен к проекту" означает, что код из этого фреймворка можно будет использовать в нашем проекте.
