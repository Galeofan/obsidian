#### На сервере сделать:
1) Открыть желаемый JMX порт, например 8999
2) Задать в параметрах запуска JVM:
```bash
-Djava.rmi.server.hostname=$SERVER_IP \ --Внешний IP или DNS сервера, чтобы RMI слушал не localhost
-Dcom.sun.management.jmxremote \
-Dcom.sun.management.jmxremote.port=$JMX_PORT \
-Dcom.sun.management.jmxremote.rmi.port=$JMX_PORT \
-Dcom.sun.management.jmxremote.local.only=false \
-Dcom.sun.management.jmxremote.authenticate=false \
-Dcom.sun.management.jmxremote.ssl=false \
```

> [!info] 
> **JMX ПОРТ 8999 (позволяет пользоваться только сэмплированием!!!)**

2) Создать подключение через remote по порту JMX
