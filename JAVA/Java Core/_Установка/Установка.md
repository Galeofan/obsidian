### Основные инструменты

1. Качаем и ставим [JDK](JAVA/Java%20Core/Базовый%20синтаксис/JDK.md) c сайта [Oracle](https://www.oracle.com/java/technologies/downloads/)
2. Качаем и ставим [Intellij IDEA Community Edition](Intellij%20IDEA/Intellij%20IDEA.md) с сайта [Jetbrains](https://www.jetbrains.com/idea/download/?section=windows)
3. Качаем и устанавливаем [GIT](GIT/GIT.md): [Тык](https://gitforwindows.org/)
4. В приложухе GitBash генерим ключи командой: `ssh-keygen -t rsa -b 4096 -C "your_email@example.com"`, где вместо `your_email@example.com` указываем свою почту
5. Командой чекаем что агент для работы через SSH запущен: eval "$(ssh-agent -s)"
6. Грузим ключи в агента SSH: ssh-add ~/.ssh/id_rsa
7. Копируем в буфер публичный ключ и грузим его в GitHub: clip < ~/.ssh/id_rsa.pub
8. Авторизуемся в Git: ssh -T git@github.com

### Команды для проверки версии Java

Для проверки версии Java можно использовать 2 команды, их вывод примерно аналогичен:

 ```bash
	ls /Library/Java/JavaVirtualMachines
 ```

```Shell
	java --version
```
